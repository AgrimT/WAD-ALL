SPPU – TE Information Technology

Savitribai Phule Pune University | Third Year IT Syllabus 2019 Pattern

📗 Semester 5 Subjects

Database Management Systems (DBMS)
Units:
1. Introduction to DBMS, ER Model
2. Relational Model, SQL
3. Normalization (1NF, 2NF, 3NF, BCNF)
4. Transaction Management & Concurrency Control
5. Indexing, Hashing, Query Optimization
Computer Networks
Units:
1. Introduction to Networks, OSI Model
2. Data Link Layer, MAC
3. Network Layer – IP, Routing
4. Transport Layer – TCP, UDP
5. Application Layer – HTTP, DNS, FTP
Theory of Computation (TOC)
Units:
1. Finite Automata – DFA, NFA
2. Regular Expressions & Languages
3. Context-Free Grammars & PDA
4. Turing Machines
5. Decidability & Complexity

📘 Semester 6 Subjects

Artificial Intelligence (AI)
Units:
1. Introduction to AI, Search Algorithms
2. Knowledge Representation
3. Expert Systems
4. Machine Learning Basics
5. Natural Language Processing
Information Security
Units:
1. Introduction to Security, CIA Triad
2. Cryptography – Symmetric & Asymmetric
3. Network Security – Firewalls, VPN
4. Web Security – SQL Injection, XSS
5. Digital Signatures & PKI
